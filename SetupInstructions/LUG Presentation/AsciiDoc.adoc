== The Atom Editor


Atom is a text editor that's modern, approachable, yet hackable to the coreâ€”a tool you can customize to do anything but also use productively without ever touching a config file.

The Atom Editor is an Open-Source editor for AsciiDoc and other languages. It provides Syntax highlighting, live previews, and other functionality. Atom Editor is available on Linux, Mac OS, and Windows. It can be downloaded from https://atom.io.

*Features*

* Cross-platform editing
* File system browser
* Built-in package manager
* Multiple panes
* Smart autocompletion
* Find and replace
